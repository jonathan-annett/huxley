# Reference Source

This directory contains the original, unmodified source code from
[8086tiny](https://github.com/adriancable/8086tiny) by Adrian Cable.

These files are **read-only reference material**. They are never modified.
The emu86 emulator in `src/` is a clean-room refactor informed by this reference.

## Files

- `8086tiny.c` — The original emulator source (to be copied here by EMU-01)
- `bios.asm` — The original BIOS assembly source (to be copied here by EMU-01)
- `LICENSE` — MIT License, copyright Adrian Cable
